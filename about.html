About

Class up your CSS with Tuxedo. Many frameworks out today provide solutions that aren’t easy to customize. Tuxedo is designed semantically - using HTML5 built in tags for easier understanding and cleaner markup.

Commented Code

Tuxedo’s code is well commented so you, the developer, know what to change and where to change it. That way, whether the changes you want to make are large or small you know just where to start. Code is designed to be manipulated - not left alone in a file full of class names and ids you may not understand and definitely don’t want to touch.

Sassy CSS

Tuxedo uses Sass with the .scss file extension. If you’ve never used Sass, just know that with the .scss extention it’s structured pretty much like CSS3, but with some awesome additional features (*ahem* variables) thrown in. Don’t worry about never having used Sass before. The Getting Started page tells you all you need to know to get it set up.

Download only what you need

With Tuxedo you can also download just the pieces you need rather than getting bogged down in all the files you don’t need. Download the full version below - or click on one of the links to download just the section you need. Best of luck!
