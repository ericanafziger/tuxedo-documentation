About Tuxedo

Class up your CSS with Tuxedo.

Many frameworks out today provide solutions that aren’t easy to customize. Tuxedo is designed semantically - using HTML5 built in tags for easier understanding and cleaner markup. Code is designed to be manipulated - not left alone in a file full of class names and ids you may not understand and definitely don’t want to touch.


Tuxedo’s code is well commented so you, the developer, know what to change and where to change it. That way, whether the changes you want to make are large or small you know just where to start.


Tuxedo uses Sass with the .scss file extension. If you’ve never used Sass, just know that with the .scss extention it’s structured pretty much like CSS3, but with some awesome additional features (*ahem* variables) thrown in. Don’t worry about never having used Sass before. The Getting Started page tells you all you need to know to get it set up. If you’re still having trouble, here is a great tutorial on getting started with Sass.


With Tuxedo you can also download just the pieces you need rather than getting bogged down in all the files you don’t need. Click on the button below to download the full version - or click on a link to download just the parts you need. Best of luck!
