Tuxedo was designed with you - the developer - in mind.

Whether you are front-end or back-end, this clearly commented and symantic framework will get your CSS up and running in no time. Don’t fret about overriding styles until you want to pull your hair out - Tuxedo tells you exactly what to change to get the results you want. Dig into our Sass files until your site is fully customized - we’ll tell you how.

Getting Started

To get started - download the full version of Tuxedo. This version comes complete with all the JavaScript and CSS you need to get your project underway.

Or you can clone the Tuxedo repo directly from GitHub. Just open up terminal and copy the following code snippet into the command line:

git clone https://github.com/ericanafziger/tuxedo-css

Now that you have your own copy of tuxedo, navigate into the tuxedo folder. To get started, we are going to watch the Sass component of your project. Now, when you start making changes it will be compiled as CSS. If you already have Sass installed on your machine ignore the first snippet below. Once you are done, open a new terminal window or tab; this Sass watch should be left running anytime you are working on the project.

gem install sass
sass --watch scss:css

Project Structure

Let’s take a look at what Tuxedo contains.

The index.html page contains the basic project structure to get your website started.

There are two files in the js folder. The tuxedo.js file has everything you need to make things like nav bars and animations work. The scripts.js file is for you to put your own code in.

There is an img folder with great high-res images from Unsplash to get your project started.

Finally, there is the scss folder full of all the components you need for your project. Unlike other frameworks, Tuxedo is design so you can directly edit these file. Code is clearly commented and easy to follow. Start off in the _global.scss folder where you can decide what colors and fonts to use. Tuxedo is designed to have several neutral colors and then one pop of color to keep your site looking beautiful.

tuxedo-css/ |--scss / | |--_animations.scss | |--_buttons.scss | |--_fonts.scss | |--_forms.scss | |--_global.scss | |--_grid.scss | |--_helpers.scss | |--_hero.scss | |--_mediaQueries.scss | |--_nav.scss | |--_reset.scss | |--_threejs.scss | |--_styles.scss | |--js/ | |--tuxedo.js | |--scripts.js | |--img | |--index.html
