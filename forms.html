Forms

If your website is in need of forms you may have found that customizing them can be really difficult. Tuxedo has taken care of most of the heavy lifting. You can use our forms straight out of the box or add custom styling to them with the well commented code. in the _forms.scss file.

For any form make sure to place your code inside a form tag for proper display and styling.

Inputs

Form inputs are the most basic of form elements. Get your form started with the input snippet below.

<label class="text-dark">Dark input example</label> <input type="email" class="input-dark" placeholder="Email Address">
<label class="text-dark">Light input example</label> <input type="email" class="input-light" placeholder="Email Address">
<label class="text-pop">Pop input example</label> <input type="email" class="input-pop" placeholder="Email Address">

Textareas

A textarea is a great way to allow your user to write more information and provide a longer answer. Tuxedo's built in features preserve your design by not allowing a user to drag a textarea to the ends of the earth. Try it out below.

<textarea class="textarea-light" rows="3" placeholder="Textarea text here"></textarea>

Selects

Selects by default are not pretty. Tuxedo has taken the select - and applied a style that matches Tuxedo's clean design. The best part is, the code is clearly commented so you know just what to change to adapt the select for you project design.

<label class="text-dark">Select menu</label> <select class="select-dark"> <option selected="true" disabled>Choose Option</option> <option>Option 1</option> <option>Option 2</option> <option>Option 3</option> </select>

Checkboxes

Checkboxes are another form element that makes it difficult to apply your own styles. Use Tuxedo's design straight out of the box or read through the commented code to change just what you need.

<label class="text-dark">Checkboxes</label> <div class="checkbox-dark"> <input type="checkbox" id="red"> <label for="red">Red</label> </div> <div class="checkbox-dark"> <input type="checkbox" id="yellow"> <label for="yellow">Yellow</label> </div> <div class="checkbox-dark"> <input type="checkbox" id="green"> <label for="green">Green</label> </div>

Radio Buttons

With radio buttons you can use Tuxedo's design straight out of the box or read through the commented code to change only what you need.

<label class="text-dark">Radio buttons</label> <div class="radio-dark"> <input type="radio" id="teal" name="radio-group" checked> <label for="teal">Teal</label> </div> <div class="radio-dark"> <input type="radio" id="maroon" name="radio-group"> <label for="maroon">Maroon</label> </div> <div class="radio-dark"> <input type="radio" id="blue" name="radio-group"> <label for="blue">Blue</label> </div>

Form Styling Helpers

Each form element comes in several different styling states. Inputs, selects, textareas, and form-divides can all be styled in dark, light, or pop. Checkboxes, radio buttons, and form-headers can all be styled in dark and light variations. Simple change the ending of their classes to see the changes. For example, change textarea-light to textarea-pop or textarea-dark to change the styling.
