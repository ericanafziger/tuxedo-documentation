Colors

Choosing a color palette is an important step in any web design. Tuxedo makes it easy with several color palette options to get you started. Tuxedo's design scheme is based around a base dark color, light color, and pop of color. Then there are several greys in the middle. To change these colors to your own custom ones, simply follow the steps below.

Creating color changes

Tuxedo's _global.scss file contains all the variables that gives your webpage color. When you open the globals file you should see the following (for our current purposes we are going to ignore everything below the $color-pop variable):



//change variables below to customize the tuxedo framework //colors $dark: #262525; //change global 'dark variable' $light: #F4F4F4; //change global 'light variable' $light-grey: #F4F4F4; $md-grey: #C8C8C8; $dark-grey: #5C5C5C; $color-pop: #2A7F62; //change global 'color-pop' (...)

If you've never used Sass before this might look a little mysterious so let's break it down. Sass allows you (the developer) to create variables that are implemented throughout your CSS. Variables are defined with a $ preceeding them. Tuxedo's variable names are descriptive so you know exactly what you're changing. Now, wherever you see $dark throughout your SCSS files, you know that is actually a placeholder for the color #262525 (or whatever color you've replace it with). So when you want to change the dark color in your project, you only have to change it in the global file to see your changes reflected throughout your project.

Color Palettes

Here are some color palettes we've put together to get your started.

Classic Tuxedo Palette

$dark: #262525; $dark-grey: #5c5c5c; $md-grey: #c8c8c8; $light: #f4f4f4; $color-pop: #2A7F62;

Professional Palette

$dark: #1C2C3C; $dark-grey: #505255; $md-grey: #D9DADB; $light: #F1F1F2; $color-pop: #FFC857;

Bright Palette

$dark: #262525; $dark-grey: #5c5c5c; $md-grey: #c8c8c8; $light: #f4f4f4; $color-pop: #CE4257;
