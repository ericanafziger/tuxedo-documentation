Hero

Hero sections have become very popular within webdesign today. Tuxedo has made it simple for you to integrate your own hero complete with background images and call to action buttons.

There is also a hero slideshow option for slideshows that are sure to capture the attention of your audience.

Circles

The circles div provides a visual respresentation to users of how many images are in the slideshow. Clicking on any of the circles will show the corresponding image. Slideshows do not need these circles to function properly so simply remove the entire circles div if you do not want them in your project.

Adding or removing images

Customizing your slideshow requires several simple steps and no JavaScript or CSS manipulation. Each image of the slideshow is given a unique class number that is sequential. These class numbers correspond to the circle classes below and both of these sequences must match for your slideshow to function properly.

To add another image simply duplicate one of the other images, change its class to the next number in the sequence, and change the src to the image of your choice. For example, if you are adding in a 6th slide it would be given class="6".

Next, find the div with the class of circles below and add a new div with the class="circle 6". You can continue this process for as many images as you would like to add.

Be sure that when rearranging images the class number sequence is in numerical order for both the images and cirlces. The first image and circle in the sequence should both be given the class active. If you remove the active class the slideshow will no longer function properly.

To remove an image simply delete the image and the circle with the corresponding numbered from both the images and circles divs respectively. Then adjust any class numbers so the numerical sequence is still in order.

Changing time between slides

The slideshow if set on a default timer of 5 seconds per slide. To change this interval navigate to the tuxedo.js file. Open the finder for your editor (probably CMD + F).
