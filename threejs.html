Three.js Backgrounds

Three.js is a 3D JavaScript library. Tuxedo has uses this library to create animated hero backgrounds for your web apps. Because these files are larger in scale, they do not come with the Tuxedo download and will need to be downloaded and added into your project independently.

The download comes with:

  • A threejs.scss file that needs to be added to your scss folder.
  • A three-minified.js and three-render.js file - both should be added to your js folder.

Now, open your styles.scss file within your scss folder. At the bottom of the file place the following import code:




@import "threejs";

Finally, add the following two script tags to the bottom of your body tag on your html page.

(...) <script src="js/three-minified.js"></script> <script src="js/three-render.js"></script> </body>
