Three.js Backgrounds

Three.js is a 3D JavaScript library. Tuxedo has uses this library to create animated hero backgrounds for your web apps. Because these files are larger in scale, they do not come with the Tuxedo download and will need to be downloaded and added into your project independently.

The download comes with:

  • A threejs.scss file that needs to be added to your scss folder.
  • A three-minified.js and three-render.js file - both should be added to your js folder.

Now, open your styles.scss file within your scss folder. At the bottom of the file place the following import code:




@import "threejs";

Finally, add the following two script tags to the bottom of your body tag on your html page.

(...) <script src="js/three-minified.js"></script> <script src="js/three-render.js"></script> </body>

Color Changes

The colors of Tuxedo Threejs objects are controlled by 5 variables. Open the three-render.js file and you will notice the only difference between these variables and the ones in your _global.scss file is the 0x preceeding each number value. 0x is equivalent to the # hex color indicator in this situation. To change the color simply change the number after the 0x.

Material Changes

If you want your sphere to have the same shiny texture as the torus knot navigate to the three-render.js file, open up the finder in your text editor, and search change sphere material. You will be directed to the commented code for //change sphere material here. Now simply swap out materialWireframe for materialShiny. You can perform the same search by substituting sphere with torus or cube.
