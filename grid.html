Grid

Tuxedo offers two grid options: a basic grid and a flex grid. The basic grid is a mobile-responsive, standard 12 column grid with the option for gutters. The flex grid uses flexbox and makes creating custom navbars and column layouts a sinch.

Basic Grid

This is a 12 column grid. Each grid can be placed inside a .container for some styled presets, or you can ditch the container if it doesn't work within your layout. A .row is needed for each set of columns. Columns declarations should add up to 12 in each row to display correctly. Columns are specified by .col-1 .col-2 .col-3 and so on.

Resize your browser window to see the responsiveness in action!

Flex Grid

With the flex grid your content will grow and shrink to fill up remaining space in the column. There are four column sizes: .f-col-1, .f-col-2, .f-col-3, and .f-col-4.

A .f-col-2 will take up twice as much space as a f-col-1 and a f-col-3 will take up three times as much space as a f-col-1. See it in action below. Add the class .responsive to your rows to use predefined media queries at the tablet and mobile sizes.

Flex Grid Helper Classes

Flexbox allows users a lot of options for positioning items within a flexed container. Tuxedo makes it easy to apply different positioning with helper classes. Simply add one of these helpers to your f-col(umn) to position. Here's a breakdown of the positioning options and their results.
* Hint: you can add these helper classes to any items within a flexed container, not just columns!

Content is placed at flex-start- the beginning of the horizontal and vertical axis - by default if no flex direction is specified.


This content is positioned on the horizontal and vertical center axis with .flex-center.


This content is positioned on the horizontal center axis, but not the vertical center axis with .flex-center-h.


Use .flex-center-v to position items on the center of the vercal axis but not the horizontal axis.


If there is only one item in a row it will stretch to take up the entire width!

Use .flex-end to position content on the end of the horizontal axis and the end of the vertical axis.


Use .flex-end-v to position items on the end of the vertical axis but the start of the horizontal axis.


This content is positioned on the end of the horizontal axis but at the beginning of the vertical axis with .flex-end-h


Use .flex-center-end to position content in the middle of the vertical axis, but the end of the vertical axis.
