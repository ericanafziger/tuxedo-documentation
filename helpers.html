Tuxedo comes with a variety of helper classes to help make your code cleaner. These classes are reusable styles that can be paired with a various predefined tuxedo elements.

Background Helpers

Background helpers allow you to add a background with a quick class addition. Simply add the class to the element you would like to give a background.

This div has a dark background

<div class="bg-dark"></div>

This div has a light background

<div class="bg-light"></div>

This div has a pop-of-color background

<div class="bg-pop"></div>

This div has a dark-grey background

<div class="bg-dark-grey"></div>

This div has a medium-grey background

<div class="bg-md-grey"></div>

This div has a transparent, dark background

<div class="bg-t-dark"></div>

This div has a transparent, light background

<div class="bg-t-light"></div>

This div has a transparent, pop-of-color background

<div class="bg-t-pop"></div>

Text Helpers

Text helpers allow you to easy style your text in specific areas without overriding other classes and ids. Just add the class to the text element you would like to give a color makeover.

Dark Text

Dark Grey Text

Medium Grey Text

Light Text

Color Pop Text

<h4 class="text-dark">Dark Text</h4> <h4 class="text-dark-grey">Dark Grey Text</h4> <h4 class="text-md-grey">Medium Grey Text</h4> <h4 class="bg-md-grey text-light">Light Text</h4> <h4 class="text-pop">Color Pop Text</h4>

Centered Text

<h4 class="text-center">Centered Text</h4>

Position Helpers

Add these classes to elements to directly change their positioning. Position helpers work great paired with navbars. Change a navbar a fixed nav by adding the .fixed class, or have it start above the content with the .relative. The .absolute position is also available and works especially well for items already within a relative container.

<nav class="nav-dark fixed"> (...) </nav> <nav class="nav-dark relative"> (...) </nav> <nav class="nav-dark absolute"> (...) </nav>

Media Query Helpers

In certain senarios it can be helpful to hide or show a block of code based on media queries. If that's where you find yourself, these are the helper classes for you. The flex extension allows elements to be display: flex rather than display: block for elements using flexbox.

//For tablet <div class="tablet-show"> Items are displayed on tablet and mobile screen sizes and hidden on desktop. </div> <div class="tablet-hide"> Items are displayed on desktop, but not on tablet and mobile screen sizes. </div> <div class="tablet-flex"> Elements are displayed as flex containers on tablet and mobile screen sizes and hidden on desktop. </div> //For mobile <div class="mobile-show"> Items are displayed on mobile screen sizes and hidden on desktop and tablet. </div> <div class="mobile-hide"> Items are displayed on desktop and tablet, but not on mobile screen sizes. </div> <div class="mobile-flex"> Elements are displayed as flex containers on mobile screen sizes and hidden on desktop and tablet. </div>

Flex Helpers

Flex helpers are designed to be paired with flexed containers. To see an example of these guys in action visit the flex grid documentation.

The flex helpers are: .flex-center, .flex-center-h, .flex-center-v, .flex-end, .flex-end-h, .flex-end-v, .flex-center-end.

<div class="f-row responsive"> <div class="f-col-1 flex-center"> Content will be centered on horizontal and vertical axes. </div> <div class="f-col-1 flex-center-h"> Content will be centered on horizontal axis. </div> <div class="f-col-1 flex-center-v"> Content will be centered on vertical axis. </div> <div class="f-col-1 flex-end"> Content will be placed at the end of the horizontal and vertical axes. </div> <div class="f-col-1 flex-end-h"> Content will be placed at the end of the horizontal axis and beginning of the vertical axis. </div> <div class="f-col-1 flex-end-v"> Content will be placed at the end of the vertical axis and beginning of the horizontal axis. </div> <div class="f-col-1 flex-center-end"> Content will be centered on the vertical axis and placed at the end of the horizontal axis. </div> </div>
