ScrollSpy

Basic Functionality

ScrollSpy is designed to show your users exactly where they are on a page, and allows for easy navigation to other sections with a simple click and smooth scroll transition. It is easy to set up as ScrollSpy dynamically adds as many or few sections as desired.

To use ScrollSpy on your site you will need to add a navigation with numbered scrollSpy ids. Each section of your page will then be given section ids with numbers that correspond to the scrollSpy ids in the navigation.

You can see ScrollSpy in action to the left of this page. Scroll Spy works well using the flex grid layout and the fixed helper class to keep it fixed at the top of the page. Checkout out the code below to get started.

<div class="container"> <div class="f-row"> <div class="f-col-1 scrollSpy"> <div class="fixed text-dark"> <h5 id="scrollSpy1" class="active">Section 1</h4> <h5 id="scrollSpy2">Section 2</h4> <h5 id="scrollSpy3">Section 3</h4> <h5 id="scrollSpy4">Section 4</h4> </div> </div> </div> <div class="f-col-4"> <div id="section1"> <p>Your content here</p> </div> <div id="section2"> <p>Your content here</p> </div> <div id="section3"> <p>Your content here</p> </div> <div id="section4"> <p>Your content here</p> </div> </div> </div> </div>
