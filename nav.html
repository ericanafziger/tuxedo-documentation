Navigation

Alright so you want a nav bar? To start off navigate to your scss folder and find the _nav.scss file. This is where the code that controls your navbar lives. Don’t touch it quite yet and lets take a look at the default nav options.

Basic Nav

The basic nav is…you guessed it…pretty basic. But it can be a good place to get started. Just copy this code into the body of your html. I’d suggest at the very top, but that’s for you to decide.

 
 
     
  • Home
    •  
  • Shop
    •  
  • Work
    •  
<nav class="nav-dark">  <header>  <img src="img/logo-thick-light.svg" alt="Tuxedo Logo"> </header> <ul class="menu-items">  <li class="f-col-1 flex-center"> <a href="#">Home</a> </li>  <li class="f-col-1 flex-center"> <a href="#">Shop</a> </li>  <li class="f-col-1 flex-center"> <a href="#">Work</a> </li>  </ul> </nav>

Mobile Nav

Use this navigation to ensure users on mobile devices will be able to easily use your navigation. This nav displays like normal until you get to tablet size. Links will disappear and be replaced by the hamburger icon. JavaScript initiation is required for the mobile nav to work properly. After pasting the code into your HTML paste the following JavaScript into the bottom of your js/tuxedo.js file where specified.
